Introduction to ipcwqrPIC package

Introduction

ipcwqrPIC is the R package to fit a new inverse-probability censoring weighted (IPCW) estimating procedure for censored quantile regression when the data are partially interval-censored that include doubly-censored (DC) data and partly interval-censored (PIC) and possibly correlated within the same cluster. Let \(T\) and \(X\) be the event time of interest and its related \(p\)-vector of covariates, respectively. Our main objective is to estimate the \(p\)-dimensional quantile coefficient vector \({\boldsymbol{\beta}}_0(\tau)\) for some \(\tau \in[\tau_L,\tau_R]\subset (0, 1)\) in the following linear quantile regression model: \[ T_i = {\bf x}_i^T {\boldsymbol{\beta}}_0(\tau) + e_i(\tau),\quad i=1, \ldots ,n, \] where \(e_i(\tau)\) is the random error whose \(\tau\)th quantile conditional on \({\bf x}_i\) equals 0. When the data are subject to partially interval-censoring, left and right endpoints of the censoring time, \(L\) and \(R\), are observed instead of \(T\) such that \(T\in(L,R)\). Note that double-censoring can also be viewed as a special case of partly interval-censoring, i.e., \(T\) is left-censored if \(L=0\) and right-censored if \(R=\infty\).

Usages

Installation of ipcwqrPIC package can be done by

devtools::install_github(repo="YejiStat/ipcwqrPIC")

or

base::require("ipcwqrPIC")

picrq() function has the following arguments:

picrq(L=L,R=R,delta=delta,x=x,tau=tau,var.estimation = "IS")

dcrq() function has the following arguments:

dcrq(L=L,R=R,T=T,delta=delta,x=x,tau=tau,var.estimation = "IS")

See the detailed description from help(picrq()) and help(dcrq()).

We first simulate univariate partly interval-censored (PIC) data with normal random error, which is similar to PIC data simulation setting of Kim et al. (2023+).

library(ipcwqrPIC)
set.seed(111)
n = 200
x1 = runif(n,-1,1)
x2 = rbinom(n,1,0.43)
x = cbind(x1,x2)
T = 2 + x1 + x2 + rnorm(n)
U = (1 - 0.25*x1)*runif(n, -6, 5)
V = U + (1 - 0.1*x2)*runif(n, 6, 20)
U = exp(dplyr::case_when(TRUE ~ T, T>V ~ V, T<U ~ -Inf))
V = exp(dplyr::case_when(TRUE ~ T, T>V ~ Inf, T<U ~ U))
delta = ifelse(U==V, 1, 0)
tau=0.3
ipcwqrPIC::picrq(L=V,R=U,delta=delta,x=x,tau=tau,var.estimation = "IS")
#>           tau coefficients       se pvalue lower bd  upper bd
#> Intercept 0.3     5.651062 0.577157      0 4.519834  6.782290
#> x1        0.3     5.661259 0.910094      0 3.877475  7.445044
#> x2        0.3     9.241869 1.366543      0 6.563444 11.920293
ipcwqrPIC::picrq(L=V,R=U,delta=delta,x=x,tau=tau,estimation = "DR",var.estimation = "IS")
#>           tau coefficients       se pvalue lower bd  upper bd
#> Intercept 0.3     5.649042 0.567730      0 4.536290  6.761793
#> x1        0.3     5.656558 0.890667      0 3.910851  7.402266
#> x2        0.3     9.240527 1.363988      0 6.567109 11.913944

We posit two estimating methods, ipcw estimating method and doubly robust ipcw estimating method, which can be conducted by specifying estimation = NULL and estimation = “DR”, respectively.

Next, we simulate univariate doubly-censored (DC) data with normal random error, which is one of DC data simulation setting of Kim et al. (2023+).

n=200; 
x1=runif(n,-1.2,1.7); x2=rbinom(n,1,0.6)
T = 1.7+x1+x2+rnorm(n)*(1-0.1*x2)
L=runif(n,-2.8,1.9); R=L+runif(n,4.2,8.1)
Y=pmin(R,pmax(T,L))
delta=case_when(
 T<L ~ 3,
 T>R ~ 2,
 TRUE ~ 1 #observed
)
L=L; R=R; T=T; delta=delta; x=cbind(x1,x2); tau=0.3
ipcwqrPIC::dcrq(L,R,T,delta,x,tau,var.estimation = "IS")

We can choose one of three weighting types, the parametric KM estimating method, Beran’s nonparametric KM estimating method, and Ishwaran’s nonparametric survival random forests estimating method which can be accessible by using wttype=“param”, wttype=“Beran” and wttype=“Ishwaran”, respectively.

Finally, we cover multivariate doubly-censored (MVDC) data with normal random error, which is one of MVDC data simulation setting of Kim et al. (2023+).

If id=NULL, we can gain the unadjusted IPCW method estimates, else if id=id and index value is put, we can get the adjusted IPCW method.

cluster=60
v=rlnorm(cluster,mean=0,sd=0.5)
k = sample(0:9*10,cluster,replace = TRUE)
m = ifelse(quantile(v, k/100) <= v & quantile(v, (k+10)/100) <= v,
             k/10 + 2, 2)
id = as.numeric(unlist(apply(cbind(1:cluster,m),1,function(a) rep(a[1],a[2]))))
n=sum(m)
x1=runif(n,-1.4,0.9); x2=rbinom(n,1,0.56)
tabid=as.vector(table(id))
vi=as.vector(unlist(lapply(tabid, function(x) rnorm(x,0,0.5))))
T = 1+x1+x2 + vi +(rnorm(n) - qnorm(tau))
L=runif(n,-3.8,2); R=L+runif(n,3.2,9.7)
x=as.matrix(cbind(x1,x2),ncol=2)
delta=case_when(
    T<L ~ 3,
    T>R ~ 2,
    TRUE ~ 1 #observed
  )
ipcwqrPIC::dcrq(L=L,R=R,T=T,delta=delta,x=x,tau=tau,wttype="Param",var.estimation="IS",id=NULL)
#>           tau coefficients       se  pvalue lower bd upper bd
#> Intercept 0.3     0.952822 0.170519 0.0e+00 0.618604 1.287040
#> x1        0.3     0.869690 0.168769 0.0e+00 0.538904 1.200477
#> x2        0.3     0.935261 0.220142 1.1e-05 0.503783 1.366740
ipcwqrPIC::dcrq(L=L,R=R,T=T,delta=delta,x=x,tau=tau,wttype="Param",var.estimation="IS",id=id,index = 1)
#>           tau coefficients       se  pvalue lower bd upper bd
#> Intercept 0.3     0.967092 0.148700 0.0e+00 0.675641 1.258544
#> x1        0.3     0.795704 0.135045 0.0e+00 0.531015 1.060393
#> x2        0.3     0.880232 0.207229 1.1e-05 0.474064 1.286401

In real data analysis, we apply our proposed method to the multivariate clustered PIC data using a phase 3 metastatic colorectal cancer clinical trial. This dataset is accessible for data(mCRC) in the PICBayes R package (Pan, 2021).

library(PICBayes)
data("mCRC")
d = with(data.frame(mCRC), data.frame(U = ifelse(y==0,R,L),
                                      V = ifelse(y==2,L,R),
                                      # Cluster weighted data
                                      id=(rep(c(table(SITE)),c(table(SITE)))),
                                      # Treatment arm: 0 = FOLFIRI alone, 1 = Panitumumab + FOLFIRI.
                                      x1= case_when(TRT_C == 0 ~ 0, #Pan et al data
                                                    TRT_C == 1 ~ 1),
                                      # Tumor KRAS mutation status: 0 = wild-type, 1 = mutant.
                                      x2= case_when(KRAS_C == 0 ~ 1,
                                                    KRAS_C == 1 ~ 0),
                                      delta = case_when(IC == 0 ~ 1,
                                                        IC == 1 ~ 0)
));
L=(log(d$U));R=log(d$V); delta=d$delta
x = cbind(d$x1,d$x2); id=d$id;  tau=0.1;
ipcwqrPIC::picrq(L,R,delta,x=x,tau=tau,var.estimation = "IS")
#>           tau coefficients       se   pvalue  lower bd upper bd
